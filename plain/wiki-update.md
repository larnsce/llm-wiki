::: {.callout-note}
A skill is a set of instructions the model follows. The page you are reading
explains the wiki-update skill in plain language. The model itself runs the
[standard version](../skills/wiki-update/SKILL.md), which the site publishes
word for word.
:::

## What it does

wiki-update is the one sanctioned way to change text that is already on a
page. Every other skill only adds. For a factual change it demands
evidence, an archived source file or a web address, and it refuses to edit
until you provide one. It then shows you the exact before and after of
every block it would touch and waits for your approval. When new
information contradicts an old claim, the old claim is not erased. It stays
on the page, struck through and marked with the date and a pointer to the
replacement, so you can read later what the wiki used to believe and why
the belief changed.

## When to use it

Use it to correct a mistake, to record that a fact has changed, or to fix
wording, links, and typos on existing pages.

## What it never does

The skill never edits without showing the exact change first, and it never
treats your silence as approval. A factual change without a source is
refused, not negotiated. Replaced claims are never silently deleted.
