# wiki-update

STUB. This skill will be the sanctioned non-append edit path (ingest stays
append-only): revise or remove an existing cited claim, requiring a source
(`ingested/` file or URL) for every factual change, showing a before/after diff and
waiting for confirmation before writing, and handling contradictions between new
information and existing cited claims explicitly. Scheduled for v2.1;
implementation lands with issue #18.
