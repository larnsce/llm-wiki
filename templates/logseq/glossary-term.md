conflicts:: The Turing Way defines repository broadly (any storage location for data, software, or publications); Glosario restricts it to version control. Adopted: the narrow version-control sense; a data archive is a data repository.

- ## Adopted definition
	- A place where a version control system stores a project's files and the full history of changes to them.
- ## Usage
	- One page per load-bearing term only; everything else stays a table row on the domain page. Page name: `glossary/tech/repository` (Logseq file name: `glossary___tech___repository.md`). The leaf stays lowercase unless it is a proper noun.
	- `alias::` carries both language forms, so linking either Repository or Repositorium resolves to this page.
	- `domain::` links the owning domain glossary; `rule::` repeats the decision from its table row (`keep-en | translate | context`).
	- `conflicts::` records which external definitions disagree and which one you adopted. It is the payoff of having the page: one decision, applied everywhere.
	- On the owning domain page, turn the term's EN cell into a link to this page.
